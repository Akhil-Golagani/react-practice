# Setting up testing in our app

1. First install React Testing library
2. Later install Jest Library
3. Installed Babel dependencies as our project is configured using babel.
4. Configure babel
5. Configure parcel config file to diasable babel default transpliation.
6. Jest configuration
7. Jsdom - Jsdom is like a browser like environment setup to run test cases
8. Install jsdom 
9. Install @babel/preset-react to make JSX work in test cases
10. Include @babel/preset-react in my babel config
11. Install @testing-library/jest-dom

React Jest Testing phases :

    - Rendering
    - Querying
    - Assertion
